# 📂 Dataset

| Property | Value |
|----------|--------|
| Dataset Name | Metro Interstate Traffic Volume Dataset |
| Original Dataset | 48,204 Records |
| Final Training Dataset | 120,000 Records |
| Input Features | 45 |
| Target Variable | Traffic Volume |

The original Metro Interstate Traffic Volume dataset was preprocessed, feature engineered, and expanded to **120,000 records using random sampling with replacement** for training the XGBoost regression model.

# 🚀 Run Locally

Clone the repository

```bash
git clone https://github.com/kondreddyvikhila/traffic-demand-prediction.git
```

Move into the project directory

```bash
cd traffic-demand-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
